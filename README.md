# نظام إدارة المناقشات (إضافة، تعديل، حذف)
==========================

## ملخص المشروع
-----------------

نظام إدارة المناقشات هو تطبيق يسمح بإضافة، تعديل، حذف المناقشات. يهدف المشروع إلى توفير بيئة آمنة ومريحة للمستخدمين لإدارة مناقشاتهم.

## هيكل المشروع
-----------------

* `app/`: هيكل المشروع الرئيسي
 + `models/`: تعريفات النماذج (tables)
 + `controllers/`: واجهات التحكم
 + `routes/`: واجهات التوجيه
 + `services/`: خدمات التطبيق
* `config/`: إعدادات المشروع
 + `database/`: إعدادات قاعدة البيانات
 + `environment/`: إعدادات البيئة
* `docker/`: هيكل المشروع باستخدام Docker
 + `docker-compose.yml`: سند Docker Compose
 + `Dockerfile`: سند Docker
* `tests/`: اختبارات المشروع

## إعداد البيئة باستخدام Docker Compose
----------------------------------------

1. تأكد من أن Docker و Docker Compose مثبتين على جهازك.
2. افتح ترمينال وانتقل إلى مجلد المشروع.
3. استخدم الأمر التالي لتشغيل البيئة باستخدام Docker Compose:
bash
docker-compose up

4. افتح متصفح ويب ومرر إلى `http://localhost:8000` لوصول إلى التطبيق.

## وحدات المشروع
-----------------

### وحدات النموذج

* `مناقشات` (discussions)
 + `id` (رقم التعريف)
 + `عنوان` (عنوان المناقشة)
 + `وصف` (وصف المناقشة)
 + `تاريخ الإضافة` (تاريخ إضافة المناقشة)
* `تعليقات` (comments)
 + `id` (رقم التعريف)
 + `مناقشة` (رابط المناقشة)
 + `وصف` (وصف التعليق)
 + `تاريخ الإضافة` (تاريخ إضافة التعليق)

### أدوار المستخدم

* `مدير` (admin)
 + يمكن للمدير إضافة، تعديل، حذف المناقشات والتعليقات.
* `مستخدم` (user)
 + يمكن للمستخدم إضافة، تعديل التعليقات.
